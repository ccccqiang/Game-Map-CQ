## 🛠️ 创建自定义路线

在 `routes/` 目录创建新的 JSON 文件：

```json
{
  "name": "我的采矿路线",
  "loop": false,
  "points": [
    {"x": 1000, "y": 2000, "label": "矿点1", "radius": 30},
    {"x": 1500, "y": 2500, "label": "矿点2", "radius": 30},
    {"x": 2000, "y": 3000, "label": "矿点3", "radius": 30}
  ]
}
```

**参数说明：**
- `x`, `y`: 航点在大地图的像素坐标
- `label`: 航点名称（可选）
- `radius`: 到达判定半径，默认30像素
- `loop`: 是否循环执行（true/false）

## ❓ 常见问题

### Q: 路线不显示？
A: 确认三点：
1. 已点击"加载路线"按钮
2. 已勾选"启用路线导航"
3. 定位系统正常工作（能看到红色玩家标记）

### Q: 航点不自动推进？
A: 检查：
- 玩家是否真正靠近航点（进入黄色虚线圈）
- 控制台是否有 "✓ 已到达: xxx" 提示
- 尝试调整 `radius` 参数（增大更容易触发）

### Q: 如何查看完整文档？
A: 阅读以下文件：
- `路线功能说明.md` - 详细使用指南
- `功能整合报告.md` - 技术实现细节

## 🎮 使用技巧

1. **循环刷资源** - 设置 `"loop": true` 自动重复路线
2. **调整采集半径** - 修改代码中的 `self.picking_radius = 30`
3. **多路线切换** - 随时从下拉框切换不同路线
4. **重置进度** - 点击"重置路线"从头开始
