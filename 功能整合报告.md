## 🔧 技术实现细节

### 1. 核心类与方法

```python
class AIMapTrackerApp:
    # 初始化
    self.routes_dir = "routes"                      # 路线目录
    self.available_routes = {}                       # 可用路线字典
    self.current_route = None                        # 当前路线
    self.current_route_index = 0                     # 当前航点索引
    self.route_planning_enabled = False              # 导航开关
    self.collected_points = set()                    # 已采集点集合
    self.picking_radius = 30                         # 采集半径
    
    # 核心方法
    load_available_routes()         # 加载所有路线文件
    select_route(route_name)        # 选择指定路线
    get_next_waypoint()             # 获取下一个航点
    check_waypoint_reached()        # 检查是否到达航点
    advance_to_next_waypoint()      # 推进到下一航点
    calculate_nearest_route()       # 贪心算法计算最近路线
    draw_route_on_display()         # 绘制路线可视化
    check_and_advance_waypoint()    # 检测并自动推进
```

### 3. 数据流程图

```
启动程序
   ↓
扫描 routes/ 目录
   ↓
加载所有 JSON 路线文件
   ↓
用户选择路线 → 点击"加载路线"
   ↓
勾选"启用路线导航"
   ↓

   ↓
计算最近的未来5个航点
   ↓
在地图上绘制:
  - 青色连线 + 箭头
  - 航点圆圈 + 序号
  - 黄色采集范围圈

检测是否到达航点 (距离 <= radius)
   ↓
是 → 自动推进到下一航点
   ↓
否 → 继续追踪当前位置
```

## 🎯 关键特性

### 1. 贪心算法路线规划

```python
def calculate_nearest_route(self, current_x, current_y, num_points=5):
    """
    从当前位置开始，每次选择最近的未访问航点
    
    优点:
    - 时间复杂度: O(n²)，适合小规模航点集
    - 动态适应：根据玩家实时位置调整
    - 简单高效：易于理解和维护
    
    应用场景:
    - 资源采集路线
    - 跑图任务
    - 巡逻路线
    """
```

### 2. 智能航点检测

```python
def check_waypoint_reached(self, current_x, current_y, waypoint):
    """
    基于欧几里得距离判断是否到达航点
    
    公式: distance = √((x₂-x₁)² + (y₂-y₁)²)
    
    可配置参数:
    - picking_radius: 默认30像素
    - 每个航点可自定义 radius
    """
```

### 3. 高性能渲染

- **视野裁剪**: 只绘制屏幕可见范围内的航点
- **跳帧兼容**: 即使跳过AI推理帧，路线仍正常显示
- **边界检查**: 防止越界绘制导致崩溃
- **抗锯齿线条**: 使用 `cv2.LINE_AA` 提升视觉效果

### 快速开始

```bash
# 1. 运行测试验证功能
python test_routes.py

# 2. 启动主程序
python main_lk.py

# 3. 在UI中操作
#    - 从下拉框选择路线
#    - 点击"加载路线"
#    - 勾选"启用路线导航"
#    - 观察地图上的青色路线
```

### 创建自定义路线

1. 在 `routes/` 目录创建新的 JSON 文件
2. 按照以下格式编写：

```json
{
  "name": "我的路线",
  "loop": false,
  "points": [
    {"x": 1000, "y": 2000, "label": "起点", "radius": 30},
    {"x": 1500, "y": 2500, "label": "中点", "radius": 30},
    {"x": 2000, "y": 3000, "label": "终点", "radius": 30}
  ]
}
```

3. 重启程序或点击"重置路线"刷新列表
