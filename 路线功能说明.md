# 路线导航功能使用说明

## 功能概述

已成功将 LKMapTools-main 的路径规划功能整合到 main_lk.py 中，实现了以下核心功能：

### ✅ 已实现功能

1. **路线加载系统** - 自动扫描 `routes` 目录下的所有 JSON 路线文件
2. **路线选择UI** - 下拉菜单选择要执行的路线
3. **实时路线可视化** - 在地图上显示青色路线连线、箭头指示、序号标记
4. **自动航点检测** - 基于距离判断是否到达航点
5. **智能路线推进** - 到达航点后自动切换到下一个
6. **最近路线规划** - 贪心算法计算从当前位置到未来5个最近航点
7. **循环路线支持** - 支持路线完成后自动重新开始
8. **采集范围提示** - 黄色虚线圈显示采集触发范围

---

## 使用方法

### 1. 准备路线文件

路线文件存放在 `routes/` 目录下，格式示例：

```json
{
  "name": "示例跑图路线",
  "loop": false,
  "notes": "路线坐标基于地图像素坐标",
  "points": [
    {
      "x": 1598,
      "y": 4153,
      "label": "节点 1",
      "radius": 30
    },
    {
      "x": 1515,
      "y": 4110,
      "label": "节点 2",
      "radius": 30
    }
  ]
}
```

**参数说明：**
- `name`: 路线名称
- `loop`: 是否循环执行（true/false）
- `points`: 航点数组
  - `x`, `y`: 航点在大地图的像素坐标
  - `label`: 航点标签（可选）
  - `radius`: 到达判定半径（像素，默认30）

### 2. 启动程序

```bash
python main_lk.py
```

### 3. 使用路线导航

1. **选择路线** - 从下拉菜单选择要执行的路线
2. **点击"加载路线"** - 加载选中的路线文件
3. **勾选"启用路线导航"** - 开启路线可视化
4. **观察地图** - 青色线条显示前进路线，数字标注航点顺序
5. **自动推进** - 当玩家靠近航点（进入黄色虚线圈）时自动切换到下一个

### 4. 控制按钮

- **加载路线** - 加载当前选择的路线
- **启用路线导航** - 开启/关闭路线显示
- **重置路线** - 从头开始重新执行当前路线

## 视觉元素

### 路线绘制样式

- **连线**: 青色实线 (BGR: 0, 255, 255)
- **箭头**: 指向下一个航点
- **航点圆圈**: 青色空心圆，半径8像素
- **序号文字**: 青色带黑色阴影，便于识别
- **采集范围**: 黄色虚线圈，半径30像素（可配置）

## 注意事项

1. **坐标系统** - 路线坐标必须与大地图像素坐标一致
2. **性能优化** - 路线绘制仅在定位成功且启用导航时执行
3. **跳帧兼容** - 即使在跳帧时也保持路线显示
4. **视野裁剪** - 只绘制视野范围内的航点，避免越界错误
5. **循环模式** - loop=true 时路线完成后自动重置

## 故障排查

### 问题1: 路线列表为空
- 检查 `routes/` 目录是否存在
- 确认 JSON 文件格式正确
- 查看控制台输出的加载信息

### 问题2: 路线不显示
- 确认已勾选"启用路线导航"
- 检查是否正确加载路线（状态标签应显示进度）
- 确认定位系统正常工作（能识别玩家位置）

### 问题3: 航点不自动推进
- 检查 `radius` 参数是否设置合理（默认30像素）
- 确认玩家实际位置与航点坐标匹配
- 查看控制台是否有"✓ 已到达"提示
